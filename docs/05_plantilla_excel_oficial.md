# Plantilla Excel Oficial

## Archivo oficial

Nombre sugerido:
`Plantilla_Entrada_V1.xlsx`

## Hojas obligatorias
- `Series`
- `Parametros`
- `Instrucciones` (opcional para usuario, útil para operación)

## Objetivo de la plantilla

- estandarizar entrada,
- reducir errores estructurales,
- permitir validación automática,
- facilitar uso por operadores.

## Hoja `Series`
Debe contener la serie de 48 periodos con:
- `PERIODO`
- `HORA`
- `CMG`
- `P_CHAR 5`

## Hoja `Parametros`
Debe contener:
- `Q_SALIDA_CAMPANARIO`
- `FECHA_PROCESO`
- `ESCENARIO`

## Estado
Esta plantilla debe ser validada con el ingeniero antes de considerarse contrato operativo definitivo.
